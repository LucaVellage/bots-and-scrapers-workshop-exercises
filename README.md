# Workshop: Developing Bots &amp; Scrapers with Python
This repository contains exercises from the Workshop 'Developing Bots &amp; Scrapers with Python' held at the Hertie School in April 2024.

### Exercise 1: Capture the Flag
The notebook is a webscraping exercise designed to practice the use of the `Beautiful Soup` library. The task is to locate and extract 40 hidden “flags” (text snippets) from [this practice website](https://hertie-scraping-website.vercel.app).

### Exercise 2: Scraping with Selenium
The notebook contains an exercise designed to practice the use of `Selenium` for browser automation and dynamic web scraping. The script collects data from the US 2022 midterm elections from the [CNN website](https://edition.cnn.com/election/2022/results/pennsylvania/senate).
